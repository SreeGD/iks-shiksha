# Homework — Project Implementation — Capstone (Senior)

Light daily tasks. Each takes 10–15 minutes.

1. **Day 1 — Capstone Overview and Project Menu** · Write 2 sentences about today's concept. (~10 min)
2. **Day 2 — Scope and Plan** · Write 2 sentences about today's concept. (~10 min)
3. **Day 3 — Research — Day 1** · Write 2 sentences about today's concept. (~10 min)
4. **Day 4 — Research — Day 2** · Write 2 sentences about today's concept. (~10 min)
5. **Day 5 — Mid-Project Critique** · Write 2 sentences about today's concept. (~10 min)
6. **Day 6 — Build / Make — Day 1** · Write 2 sentences about today's concept. (~10 min)
7. **Day 7 — Build / Make — Day 2** · Write 2 sentences about today's concept. (~10 min)
8. **Day 8 — Document** · Write 2 sentences about today's concept. (~10 min)
9. **Day 9 — Rehearse Presentation** · Write 2 sentences about today's concept. (~10 min)
10. **Day 10 — Project Share Day** · Write 2 sentences about today's concept. (~10 min)

## Notes

- Homework is intended to consolidate what was learned in class, not introduce new content.
- If a child struggles, the parent should consult the teacher rather than push.
- Capstone project days (Module 15) may have lighter daily homework so students can focus on the project.
