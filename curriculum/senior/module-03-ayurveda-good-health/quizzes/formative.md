# Module 3 (Senior) — Formative Quizzes

## Part A — Day 3 (6 Q, 10 min)

1. (1) Which Veda is Ayurveda traditionally an Upaveda of?
2. (1) Name 3 of the 8 branches.
3. (2) Name the 3 foundational texts (bṛhattrayī).
4. (1) Caraka defines āyu (life) as a conjunction of which four components?
5. (2) Name the 3 pramāṇas (means of valid knowledge) Caraka articulates.

### Key
1. Atharva Veda. 2. Any 3 of: Kāya-cikitsā, Bāla-cikitsā, Graha-cikitsā, Ūrdhvāṅga-cikitsā, Śalya-tantra, Daṁṣṭra-cikitsā, Jarā-cikitsā, Vṛṣa-cikitsā. 3. Caraka Saṃhitā, Suśruta Saṃhitā, Aṣṭāṅga Hṛdaya (Vāgbhaṭa). 4. śarīra (body) + indriya (senses) + sattva (mind) + ātman (self). 5. Āpta-upadeśa, Pratyakṣa, Anumāna.

## Part B — Day 5 (12 marks, 15 min)

1. (2) Tridosha sub-types: name the 5 sub-types of any ONE dosha.
2. (2) Prakṛti vs Vikṛti — distinguish.
3. (2) Patwardhan (2014) coined what term to describe combining Ayurvedic typology with modern omics?
4. (2) Name 3 Ayurvedic interventions with STRONG modern evidence.
5. (2) Name 2 with WEAK evidence.
6. (2) Briefly state the 3 positions on Ayurveda's integration into modern healthcare.

### Key
1. (e.g. Vāta): Prāṇa, Udāna, Samāna, Vyāna, Apāna. 2. Prakṛti = inborn constitution; Vikṛti = current state. 3. "Ayurgenomics." 4. e.g. turmeric/curcumin anti-inflammatory, time-restricted eating (dinacharya), yoga for HRV. 5. e.g. pulse diagnosis (nāḍī parīkṣā), specific dosha-based prescriptions for unrelated conditions. 6. (A) Full integration (AYUSH); (B) Selective integration (Patwardhan); (C) Separation (cultural-historical study, biomedicine for treatment).
