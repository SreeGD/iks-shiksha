# Activity 01 — Frameworks Comparison Table

**Module:** Pañcabhūta (Senior) · **Day:** 4 · **Time:** 25 min within Day 4's 45

## Goal

Students fill in a structured comparison table contrasting Sāṅkhya, Greek (Empedocles + Aristotle), and modern frameworks across 7 dimensions.

## Materials

- Printed comparison worksheet (one per student)
- Reading extracts on hand

## Instructions

> Working alone first, then with a buddy, fill in the table comparing the three frameworks.
>
> For each cell, write a one-line answer. Where the framework has no equivalent, write "—" and we'll discuss why in the next session.

### Table (student-facing)

| Dimension | Sāṅkhya | Greek (Empedocles + Aristotle) | Modern science |
|-----------|---------|----------------------------------|------------------|
| Number of elements | | | |
| Origin (eternal? emerge? created?) | | | |
| What drives combination | | | |
| Includes empty space? | | | |
| Includes consciousness? | | | |
| Perceptual mapping? | | | |
| Method (observation, experiment, philosophy)? | | | |

### Discussion (last 10 min)

After 15 min of filling in: pair with a buddy and compare. Where do you disagree? Why?

Teacher asks: which row revealed the most interesting cross-cultural difference?

## Differentiation

- **Tier up:** Add an 8th row of your own choosing.
- **Tier down:** Focus on the first 4 rows; the others can be discussed orally.

## Clean-up

- Tables collected — used in the Day 8 paper outline.
