## Glossary

| Sanskrit | Meaning |
|----------|---------|
| tattva | principle, "thatness" |
| prakṛti | primordial nature |
| puruṣa | consciousness, witness |
| guṇa | quality (Sāṅkhya: sattva, rajas, tamas) |
| sarga | manifestation, creation |
| avyakta / vyakta | unmanifest / manifest |
| bhūta / mahābhūta | gross element |
| tanmātra | subtle perceptual essence |
| indriya | sense organ |
| ākāśa | space / ether |
| śabda, sparśa, rūpa, rasa, gandha | sound, touch, form, taste, smell |
