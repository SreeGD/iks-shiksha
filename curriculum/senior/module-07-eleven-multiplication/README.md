# ×11 Multiplication — Ekādhikena Pūrveṇa

**Band:** Senior (Grades 9–12) · **Sessions:** 10 × 45 min · **Sanskrit:** Ekādhikena Pūrveṇa

## Overview

A mental-arithmetic method for multiplying any number by 11 instantly: split the digits, add adjacent pairs, place sums in between. Tirthaji's *Vedic Mathematics* (1965) labelled this an application of the *Ekādhikena Pūrveṇa* sūtra. Honest history is taught alongside: this is a 20th-century mental-arithmetic system, not strictly ancient mathematics.

## Key terms (this module)

- *Ekādhikena Pūrveṇa* — "by one more than the previous" — sūtra 1 in Tirthaji
- *sūtra* — aphorism / formula
- *Anurūpyena* — "proportionately" — sūtra used for ×12, ×13 extensions

## 10-day map

- **Day 1** — The Hook — What Is the Trick?: Compute 23×11, 35×11 mentally using the split-and-sum method.
- **Day 2** — The Carry Case: Compute 87×11, 65×11 correctly using the carry rule.
- **Day 3** — Why It Works — The Algebra: Derive (10a+b)×11 = 100a + 10(a+b) + b and explain.
- **Day 4** — 3-Digit × 11: Compute 234×11 = 2_5_7_4 using the extended split method.
- **Day 5** — Speed Day: Complete a 50-problem mental-math sprint in 5 minutes.
- **Day 6** — Anurūpyena — ×12 to ×19: Extend the method to ×12 and ×13 using one-more-digit adjustments.
- **Day 7** — Limits of the Method: Recognise when traditional long multiplication is faster than the sūtra method.
- **Day 8** — Tirthaji and Historical Context: Explain who Tirthaji was and why scholars debate the "Vedic" attribution.
- **Day 9** — Mixed Practice: Solve a mixed problem set using best-fit method per problem.
- **Day 10** — Final Assessment: Take the summative quiz and reflect on which method works best.
