# Subtraction — Nikhilam (All from 9)

**Band:** Senior (Grades 9–12) · **Sessions:** 10 × 45 min · **Sanskrit:** Nikhilaṁ Navataścaramaṁ Daśataḥ

## Overview

*Nikhilaṁ Navataścaramaṁ Daśataḥ* — "all from 9, the last from 10" — is Tirthaji's second sūtra. It produces fast complements to powers of 10, removing the need for borrowing in many subtractions. Conceptually identical to two's-complement binary arithmetic used in modern computers.

## 10-day map

- **Day 1** — The Sūtra — All from 9, Last from 10: Compute the complement of any 2-digit number to 100.
- **Day 2** — Complement to 1000 and Beyond: Compute complements to 1000 and 10000 mentally.
- **Day 3** — Subtract from a Power of 10: Compute 1000−567 and similar instantly.
- **Day 4** — General Subtraction via Complements: Use add-the-complement trick for arbitrary subtractions.
- **Day 5** — Why It Works — The Algebra: Derive the place-value identity that makes the rule work.
- **Day 6** — Bar Model Proof: Draw a visual proof using bar models and number lines.
- **Day 7** — Two's Complement in Computers: Show how the same idea underlies binary computer subtraction.
- **Day 8** — Speed Day: Complete a 30-problem mental subtraction sprint.
- **Day 9** — Mixed Practice: Solve a mixed problem set comparing complement vs traditional borrow.
- **Day 10** — Final Assessment: Take the summative quiz.
