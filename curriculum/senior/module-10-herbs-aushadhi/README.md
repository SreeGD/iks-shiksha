# Herbs — Auṣadhi

**Band:** Senior (Grades 9–12) · **Sessions:** 10 × 45 min · **Sanskrit:** Auṣadhi

## Overview

Six widely-available herbs — Tulsī, Neem, Turmeric, Ginger, Ajwain, Mint — studied through botanical identification, traditional uses from Āyurveda, modern phytochemistry (active compounds), and safety. The module emphasises that kitchen amounts are safe, but therapeutic doses require a qualified practitioner.

## Key terms (this module)

- *auṣadhi* — medicinal plant / herb
- *tulasī* — holy basil — *Ocimum tenuiflorum*
- *nimba* — neem — *Azadirachta indica*
- *haridrā* — turmeric — *Curcuma longa*
- *ārdraka* — ginger — *Zingiber officinale*
- *yavānī* — ajwain — *Trachyspermum ammi*
- *pudīnā* — mint — *Mentha* spp.

## 10-day map

- **Day 1** — What is Auṣadhi?: Define auṣadhi and recognise the six herbs of this module.
- **Day 2** — Tulsī — Holy Basil: Identify tulasī by leaf and smell; name the active compound (eugenol).
- **Day 3** — Neem — Bitter Healer: Identify nimba; learn about azadirachtin; understand high-dose safety warning.
- **Day 4** — Turmeric — Golden Root: Distinguish fresh root vs powder; identify curcumin as active compound.
- **Day 5** — Ginger — Warming Root: Distinguish fresh vs dried; gingerol vs shogaol.
- **Day 6** — Ajwain — Carom Seed: Recognise seeds and leaves; identify thymol; traditional digestive use.
- **Day 7** — Mint — Cooling Leaf: Recognise several Mentha varieties; identify menthol.
- **Day 8** — Herbs and the Doṣas: Categorise the six herbs by heating/cooling effect (links to Module 4).
- **Day 9** — Build the Class Deck: Each student contributes one identification card to the class library.
- **Day 10** — Final Share and Assessment: Present the class herb deck; take summative quiz.
