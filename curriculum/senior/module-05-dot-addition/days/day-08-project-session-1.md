# Day 8 — Project Session 1: Choose Track + Outline

**Module:** Bindu Paddhati (Senior) · **Time:** 45 min

## Learning objective

Each student chooses Math track OR History track, drafts a one-paragraph thesis, and writes a working outline.

## Materials

- Project brief
- Rubric
- Drafting materials

## Flow

### Warm-up (5 min)

- Hand out rubric. Note: tracks have different rubric weighting (see project brief).

### Track choice + planning (15 min)

- Each student commits to a track:
  - **Math track:** Generalise the dot method to base-*n* and prove correctness. Optional extension: compare complexity with standard algorithm.
  - **History track:** Argue a position on the labelling / provenance of "Vedic mathematics."

- Write down: working title + thesis paragraph + 3 sources (must include 1 from Tirthaji, 1 from Dani, 1 from another).

### Outline drafting (15 min)

Standard 5-6 page paper outline:
- Introduction with thesis
- Section 1
- Section 2
- Section 3
- Counter-argument + response
- Conclusion

### Begin drafting (10 min)

Start the introduction. Don't worry about perfection.

## Homework

- Continue drafting at home. Aim for 1.5–2 pages by tomorrow.

## Differentiation

- **Tier up:** Math track — propose your own algorithm extension (e.g. to multiplication using the same dot logic).
- **Tier down:** History track — narrow to a single quote from Tirthaji and analyse it deeply.

## Teacher notes

- Math-track students need access to LaTeX or clean number-formatted writing for proofs.
- History-track students need ALL three sources (Tirthaji preface, Dani, Plofker) at hand.
