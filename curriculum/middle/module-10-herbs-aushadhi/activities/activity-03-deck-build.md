# Activity 03 — Herb Deck Build

**Module:** Auṣadhi · **Band:** Middle · **Day:** 9 (and finalised Day 10 morning) · **Time:** 35 min in-class + variable at home

## Type

Project artefact construction, paired

## Goal

Each pair produces six A6 (or quarter-A4) cards — one per module herb — with all eight required elements per card. The deck becomes the pair's project submission and a classroom resource after Day 10.

## Materials per pair

- 8 blank cards (6 + 2 spares)
- Colour pencils, fine-tip markers, ruler
- Reference materials (their own Days 2–8 worksheets)
- 1 envelope or rubber band for storing the finished deck

## The card template (eight elements per card)

```
+-------------------------------------+
|  [ SKETCH OF HERB ]                 |
|                                     |
|  Sanskrit (IAST):  ______________   |
|  English / Common:  ______________  |
|  Botanical name:    ______________  |
|  Family:            ______________  |
|                                     |
|  Identifying features:              |
|    1. ____________________________  |
|    2. ____________________________  |
|    3. ____________________________  |
|                                     |
|  Active compound:   ______________  |
|                                     |
|  Traditional use:   ______________  |
|  Safety note:       ______________  |
+-------------------------------------+
```

## Instructions (student-facing)

> **Today: build the deck.**
>
> 1. Lay out your six blank cards. Decide who draws which (3 each, then swap and review).
> 2. Fill in the eight elements per card. Use your worksheets — don't re-invent.
> 3. **Sketch first** (least time-consuming), **safety note last** (most time-consuming to get right).
> 4. Check each card against the rubric before finalising. The rubric is on the wall.
> 5. By the end of class, all six cards should be drafted. Refine at home tonight.

## Quality checklist for each card (rubric-aligned)

- [ ] Sketch is clear enough that someone who's never seen the herb could recognise it
- [ ] Sanskrit name in IAST (with diacritics)
- [ ] Botanical name italicised, correctly spelled
- [ ] Family named (one of: Lamiaceae, Apiaceae, Zingiberaceae, Meliaceae)
- [ ] Three distinct identifying features (not three versions of the same feature)
- [ ] Active compound correctly named (eugenol, azadirachtin, curcumin, gingerol, thymol, menthol)
- [ ] One specific traditional use (no vague claims)
- [ ] One specific safety note (no vague reassurances)

## Common failure modes (teacher watches for these)

1. **Sketching one card for 20 minutes.** Cap at 5 minutes per sketch.
2. **Vague traditional use** ("good for health"). Demand specificity: *"good for what — relieving congestion? helping digestion? cooling in summer?"*
3. **Vague safety note** ("be careful"). Demand specificity: *"be careful of what — concentration? dose? interactions?"*
4. **Two partners not coordinating.** If you see this — have them switch sides at minute 15.

## Differentiation

- **Tier up:** A 7th bonus card (any herb), applying the same eight-element template — *aśvagandhā*, *brāhmī*, *triphalā*, *kariveppilai*, *dhānyaka*. Plus: write a 30-second pitch for the bonus card on the back.
- **Tier down:** Reduce to 4 cards (tulsi, ginger, turmeric, mint) but each must have all 8 elements complete. Quality over quantity.

## Looks-like / sounds-like

| | Looks like | Sounds like |
|---|---|---|
| Going well | Both partners drawing/writing; checking the rubric on the wall; quick discussion before each entry | "What's the safety note for ginger?" / "Heartburn at high doses?" / "Yes — write that." |
| Needs intervention | One partner doing everything; cards being copied verbatim from worksheets without engagement | Silence; or "let's just write 'good for cold'" |

## Clean-up

- Each pair stores their cards in an envelope with both names + class on the front.
- Refine at home overnight; bring on Day 10.
