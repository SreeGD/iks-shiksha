# Module 2 — Panchabhūta (Five Elements)

**Age band:** Middle · **Grades:** 6–8 · **Duration:** 10 sessions × 45 min (≈ 2 weeks)

## Learning objectives

By the end of this module, students will be able to:

1. **Recall** the five great elements (*pañca-mahābhūta*) — *pṛthvī*, *ap*, *tejas*, *vāyu*, *ākāśa* — with one observable example of each.
2. **Explain** the Sāṅkhya creation sequence (space → air → fire → water → earth) as a nested accumulation of qualities, using a diagram.
3. **Distinguish** between the *bhūta*–*tanmātra* mapping (each element associated with a perceptual quality: sound, touch, form, taste, smell) and the modern atomic/molecular framework.
4. **Apply** the five-element lens to classify everyday objects and observe imbalances in a local ecosystem (e.g. air pollution, water scarcity).
5. **Compare** the *pañcabhūta* framework with the modern states-of-matter framework, noting what each scheme captures that the other does not.

## Prerequisites

- Familiarity with three states of matter (solid, liquid, gas) from middle-school science.
- Basic five senses vocabulary.
- No Sanskrit knowledge assumed; all terms introduced with IAST + meaning.

## Key vocabulary

See `_shared/glossary.md` for full entries.

- *pañca-mahābhūta* — five great elements
- *pṛthvī, ap, tejas, vāyu, ākāśa* — earth, water, fire, air, space
- *tanmātra* — subtle perceptual essence (introduced lightly at Middle band)
- *guṇa* — quality (used loosely as "property" at this band)

## Materials checklist (whole module)

- 5 sample objects representing each element (rock, water jar, candle/lamp, balloon, empty box)
- Chart paper or pinboards for the element diagram
- Index cards (~60) for the sorting activity
- Smartphones or paper for the home-observation walk (Day 4 / homework)
- Project supplies depending on project format (poster paper, recyclables, audio recorder, etc.)

## Project brief (one-line teaser)

> Each student (or pair) chooses one element and produces a poster, demo, short story, or 90-second video that shows **(a)** what the *pañcabhūta* tradition says about it, **(b)** one modern scientific parallel, and **(c)** one example of its imbalance in the student's own neighbourhood.
