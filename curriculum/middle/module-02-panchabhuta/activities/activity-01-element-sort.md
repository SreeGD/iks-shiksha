# Activity 01 — Element Sort

**Module:** Panchabhūta · **Band:** Middle · **Day:** 4 · **Time:** 25 min (within Day 4's 45 min)

## Type

Hands-on classification, paired

## Goal

Each pair sorts 5 real objects into the five-element framework and documents which other elements are also present in each.

## Materials per pair

- 3 objects from each student's home (5–6 objects per pair after duplicates)
- 2 mystery objects supplied by the teacher per pair (e.g. a stretched balloon, a glass of cloudy water, a battery, a piece of camphor, a chunk of beeswax, a small mirror)
- 1 printed worksheet (next page)
- 1 pen each

## Set-up (teacher, before class)

- Lay out 5 tray-areas around the room, each clearly labelled with one element name (Sanskrit + English).
- Distribute 2 mystery objects to each pair as they enter.

## Instructions (student-facing — printable handout)

> **Element Sort** — work in pairs.
>
> 1. Lay out your 5 objects on your desk.
> 2. For each object, decide which element is **dominant** (the strongest match). Place the object on the matching element tray.
> 3. Fill in the worksheet table below for all 5 objects.
> 4. **Disagree with your partner?** Argue it out before you fill in the row. If you can't agree, write both answers and circle the one your pair finally votes for.
> 5. **Gallery walk (last 5 min):** visit two other pairs' trays. If you disagree with where they placed an object, write a sticky note explaining why (kindly).

## Discussion prompts (last 5 min, whole class)

- Which object was the hardest to classify? Why?
- Was there an object that two pairs put on different trays? What was the strongest argument for each side?
- Did any object seem to fit on *no* tray? What does that tell us?

## Looks-like / sounds-like

| | Looks like | Sounds like |
|---|---|---|
| Going well | Both partners are pointing at the object, talking it through, sometimes disagreeing then resolving | "What about the smell?" / "Yes but the form is more obvious" / "Let's say dominant is *pṛthvī* but *gandha* is the strongest tanmātra" |
| Needs intervention | One partner doing all the writing while the other sits silently; rows filled in with one-word answers and no thought | Silence; "I don't know, write whatever" |

## Safety / sensitivity

- No fire / no flame in this activity (despite *tejas* — students can find heat sources without lighting anything).
- No food sharing for tasting; if a student wants to demonstrate *rasa*, they taste their own food, not a classmate's.
- Allergy check before allowing mystery objects like peanut shells, citrus peel, etc.

## Differentiation

- **Tier up:** After sorting, the pair picks their single "most mixed" object and writes a paragraph allocating percentages to each element (e.g. 50% *pṛthvī*, 30% *ap*, 15% *vāyu*, 5% *tejas*, 0% *ākāśa*). Defend the percentages.
- **Tier down:** Sort first into THREE categories (*pṛthvī, ap, vāyu*) and come back to add *tejas* and *ākāśa* once the first three are confident.

## Clean-up

- Each pair takes their own objects home.
- Teacher resets the 5 trays for the next class.
- Worksheet collected (stored in the project folder for Day 8 reference).
