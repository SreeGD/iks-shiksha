## Day 2 — The Creation Sequence

The five elements emerge in this order:

```
ākāśa  →  vāyu  →  tejas  →  ap  →  pṛthvī
(space)   (air)    (fire)   (water)  (earth)
```

**The quality-count rule:** each element inherits the qualities of the elements before it, then adds one new quality of its own.

| Element | Number of qualities |
|---------|---------------------|
| *ākāśa* | 1 |
| *vāyu*  | 2 |
| *tejas* | 3 |
| *ap*    | 4 |
| *pṛthvī*| 5 |

## Day 3 — Five Qualities and Five Senses

The *tanmātras* are the qualities that "belong" to each element. Each tanmātra is perceived by one sense organ.

| Element | Tanmātra (quality) | Sense organ |
|---------|---------------------|---------------|
| *ākāśa* | *śabda* (sound) | ear |
| *vāyu* | *sparśa* (touch) | skin |
| *tejas* | *rūpa* (form / colour) | eye |
| *ap* | *rasa* (taste) | tongue |
| *pṛthvī* | *gandha* (smell) | nose |
