# Module 2 (Middle) — Formative Quizzes

Two parts: Part A on Day 3 (after qualities lesson), Part B on Day 5 (mid-module). Both are diagnostic — not graded. Hand back within 24 hours.

---

## Part A — Day 3 (3 min, 5 questions)

**Name:** ______________________  **Class:** ______

**1.** Write the Sanskrit name for the **water** element.

_______________________

**2.** Which sense organ perceives *rūpa* (form / colour)?

a) ear
b) skin
c) eye
d) tongue

**3.** Fill in the blank: "The *tanmātra* of *vāyu* is _______."

**4.** True or false: *Ākāśa* has only one quality, which is sound.   (T / F)

**5.** Name ONE object in this room dominated by *tejas*.

_______________________

---

### Part A — answer key

| Q | Answer | Notes |
|---|--------|-------|
| 1 | *ap* (or *jala*) | Either is acceptable; *ap* preferred. |
| 2 | c) eye | — |
| 3 | *sparśa* (touch) | Spelling tolerated if recognisable. |
| 4 | T | — |
| 5 | tube light / sunlight through window / heater / candle / etc. | Any source of heat or light. |

## Part B — Day 5 (10 min, 5 questions, mixed format)

**Name:** ______________________  **Class:** ______

**1.** (2 marks) Write the five elements in the order they emerge in the Sāṅkhya creation sequence.

___________ → ___________ → ___________ → ___________ → ___________

**2.** (3 marks) Complete the quality table.

| Element | Number of qualities |
|---------|---------------------|
| *ākāśa* | _____ |
| *vāyu*  | _____ |
| *tejas* | _____ |
| *ap*    | _____ |
| *pṛthvī*| _____ |

**3.** (2 marks) Match each element to its *tanmātra*. Draw lines.

```
ākāśa        gandha (smell)
vāyu         śabda (sound)
tejas        rasa (taste)
ap           sparśa (touch)
pṛthvī       rūpa (form)
```

**4.** (1 mark) Name one object in this room dominated by *ākāśa*.

___________________________________________________________

**5.** (2 marks, short answer) Why does it make sense that *pṛthvī* has all five qualities?

___________________________________________________________

___________________________________________________________

---

### Part B — answer key

| Q | Answer | Mark scheme |
|---|--------|-------------|
| 1 | ākāśa → vāyu → tejas → ap → pṛthvī | 2 if all 5 in correct order; 1 if 3–4 correct; 0 otherwise. |
| 2 | 1, 2, 3, 4, 5 | 3 if all correct; 2 if 3–4 correct; 1 if 1–2 correct. |
| 3 | ākāśa↔śabda, vāyu↔sparśa, tejas↔rūpa, ap↔rasa, pṛthvī↔gandha | 2 if all 5; 1 if 3–4. |
| 4 | The empty space between two students; the air-gap above their head; the gap in a doorway; any answer that names a *void* not a thing. | 1 if it names empty space rather than an object. |
| 5 | Earth is the last element to emerge, so it inherits all qualities from the elements before it (sound, touch, form, taste) and adds its own (smell). You can perceive a piece of soil with every sense. | 2 if the answer mentions both *inherits previous* AND *adds smell*. 1 if either part is present. |

Total Part B: 10 marks. Not added to summative.

---

## What to do with the results

- A class average below 6/10 on Part B = significant gaps. Reteach the quality-count rule and tanmātra map on Day 6 (push the cross-disciplinary content to Day 7).
- Individual students below 3/10 on Part B = pull aside on Day 7 for a 10-min one-on-one before group activity starts.
