## Day 3 — Both Below 100

**The procedure (base 100):**
1. Find each factor's deficit from 100.
2. Cross-subtract for LEFT.
3. Multiply deficits for RIGHT.
4. Pad RIGHT to 2 digits.
5. Carry overflow into LEFT.

| Problem | Deficits | Left | Right (padded) | Answer |
|---------|----------|------|----------------|--------|
| 97 × 96 | 3, 4 | 93 | 12 | 9312 |
| 99 × 98 |       |   |   |    |
| 95 × 94 |       |   |   |    |
| 88 × 89 |       |   |   |    |

## Day 5 — Mixed Cases

**The change:** one above, one below. Negative RIGHT part.

| Problem | Signed | Left | Right (signed) | Answer |
|---------|--------|------|----------------|--------|
| 102 × 98 | +2, −2 | 100 | −4 | 9996 |
| 104 × 96 |          |     |     |       |
| 108 × 95 |          |     |     |       |

**Notice:** When excess = deficit, LEFT = base. Then answer = base² − (deficit × deficit).
