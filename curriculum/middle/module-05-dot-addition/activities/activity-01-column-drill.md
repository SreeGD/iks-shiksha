# Activity 01 — Column Drill (Multi-column)

**Module:** Bindu Paddhati · **Band:** Middle · **Day:** 4 · **Time:** 25 min

## Type

Paired practice with self-correction

## Goal

Each student solves 8 multi-column addition problems using the dot method, with the partner checking accuracy.

## Materials per pair

- 1 printed problem sheet (8 problems, see below)
- 1 stopwatch
- 1 pen each

## Set-up (teacher, before class)

- Print the problem sheet (you can vary the numbers — keep difficulty graded from 3×3-digit to 5×4-digit).

## Instructions (student-facing handout)

> **Multi-column Drill** — work in pairs.
>
> 1. Both partners solve all 8 problems individually (8 min). Don't look at each other's work.
> 2. Now swap sheets. Partner-A reads problem 1 aloud; partner-B re-solves it on a fresh sheet (no peeking at partner-A's answer). Compare answers. If they differ, work out which is correct.
> 3. Repeat for problems 2–8, alternating who reads.
> 4. Last 5 min: each student re-does any 3 problems they got wrong, silently.

### Problem sheet

```

     356            342            718            274
   + 198          + 925          + 386            395

[Answers (teacher copy):
```

## Discussion prompts (last 2 min)

- Which problem took the longest? Why?
- Did your partner's solution differ from yours on any problem? Whose was right?
- Were the carries the most error-prone part?

## Looks-like / sounds-like

| | Looks like | Sounds like |
|---|---|---|
| Going well | Pair is comparing answers, pointing at specific digits, debating | "I got 14702, you got 14701 — where's the difference?" |
| Needs intervention | One partner just copies the other's answers | Silence; "fine, whatever" |

## Safety / sensitivity

- No timed competition between pairs publicly. Pairs are self-pacing.

## Differentiation

- **Tier up:** 5×5-digit problems (printed on a separate sheet).
- **Tier down:** 3×3-digit problems only; build up.

## Clean-up

- Worksheets collected (for the teacher to spot-check before Day 5).
