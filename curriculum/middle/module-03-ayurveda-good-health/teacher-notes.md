# Teacher Notes — Module 3 (Middle)

## Posture

The honest framing:

> **Ayurveda is a real, ancient, sophisticated framework of lifestyle medicine. It is well-suited for: prevention, chronic conditions, daily wellbeing, dietary guidance. It is NOT a substitute for modern medicine for: acute illness, infections, surgery, life-threatening conditions.**

The "both/and" framing is critical. We are NOT teaching that Ayurveda replaces modern medicine. We ARE teaching that the Ayurvedic framework has real wisdom about sleep, food, daily rhythms, and that modern medicine increasingly validates many of these (circadian biology, gut microbiome, sleep science).

## Hard rule — never teach as medical advice

The Day 3 self-assessment is **fun and educational**, NOT diagnostic. Make this explicit:
- "This is not a doctor's diagnosis."
- "If you are unwell, see a qualified physician."
- "Some students take medications; nothing in this module changes that."

If a student says "but I think I have a Pitta imbalance" — gently redirect: "the framework is interesting; if you have actual health concerns, talk to a doctor."

## Common misconceptions

1. **"Ayurveda doesn't work."** False at the population-medicine level. Many Ayurvedic herbs (turmeric, ashwagandha, neem) have modern evidence. Many dinacharya practices (regular sleep, balanced meals, oil massage) align with modern lifestyle medicine.
2. **"Ayurveda works for everything."** False at the acute-care level. If you have pneumonia, you need antibiotics, not turmeric.
3. **"Dosha types are scientifically validated."** Mixed evidence. The tridosha framework is best understood as a *typology of tendency*, not a strict biomedical classification. Some studies show correlates; many do not.
4. **"Sushruta did rhinoplasty."** True. Documented in Suśruta Saṃhitā. European surgeons learned the technique via 18th-c. Bengali practitioners. (This is a Module 1 fact; reinforces Ayurveda's scope.)

## On the religious dimension

Ayurveda has theological associations (Dhanvantari, the deity of Ayurveda; the Caraka Saṃhitā's invocations). The classroom frame: respect the tradition's religious dimensions, focus on the lifestyle and pharmacological insights, don't conflate.

## Differentiation

- **Tier up:** Read excerpts from Caraka Saṃhitā chapter on dinacharya.
- **Tier down:** Focus on Vāta-Pitta-Kapha + 3 dinacharya practices.

## Project oversight

- The 7-day dinacharya project is personal. Students should NOT share specifics they're uncomfortable with.
- Strong projects have *realistic* schedules (not "wake at 4am every day if you've never done this").

## If you only have 6 days

- Day 1: Ayurveda + Swasthya
- Day 2: Tridosha
- Day 3: Dinacharya
- Day 4: Ahāra + BG 6.17
- Day 5: Project session
- Day 6: Presentations + summative
