# Day 5 — Mid-Module Check + Project

**Module:** Ayurveda (Middle) · **Time:** 45 min

## Learning objective

Students consolidate Days 1–4 + commit to project (their 7-day dinacharya design).

## Materials

- Formative quiz Part B
- Project brief

## Flow

### Warm-up (3 min)

Daily prompt.

### Formative quiz (15 min)

`quizzes/formative.md` Part B. Closed notebook.

### Review (10 min)

Walk through quiz; common errors.

### Project commitment (15 min)

Hand out project brief.

Each student decides:
- Target audience for their dinacharya (themselves, a sibling, a parent, a peer with a busy schedule).
- Realistic time slots.
- One specific practice they're committing to from each block: morning, midday, evening, night.

Submit project plan paragraph.

### Wrap-up (2 min)

- Preview Day 6: *"Tomorrow — ahāra. Food."*

## Homework

- Submit project plan paragraph.
