# Activity 01 — Design Your Dinacharya

**Module:** Ayurveda (Middle) · **Days:** 8 · **Time:** 30 min within Day 8

## Goal

Each student drafts a personalised 7-day dinacharya with Ayurvedic justification per item.

## Materials per student

- Worksheet (below)
- Pencil

## Worksheet template

```
Name: ______________________

DAILY ROUTINE (Mon–Fri, school days):

| Time | Practice | Ayurvedic principle |
|------|----------|----------------------|
| Wake (___): _______ | ______________ | ______________ |
| Morning hygiene: _______ | ______________ | ______________ |
| Pre-school: _______ | ______________ | ______________ |
| Mid-morning meal: _______ | ______________ | ______________ |
| Midday meal: _______ | ______________ | ______________ |
| Afternoon: _______ | ______________ | ______________ |
| Dinner: _______ | ______________ | ______________ |
| Pre-bed wind-down: _______ | ______________ | ______________ |
| Sleep: _______ | ______________ | ______________ |

WEEKEND DIFFERENCES (Sat/Sun):
__________________________________________________________________

REALISM CHECK:
- Is this achievable for my actual life? Y / N
- If "N" on any item, what's a smaller version I can do? __________________________________________________________________
```

## Discussion (5 min)

- Whose dinacharya is the most ambitious? Whose is the most realistic?
- Which item was hardest to find a Ayurvedic principle for?

## Differentiation

- **Tier up:** Add a Ritucharya (seasonal routine) layer — what would change in summer vs. winter?
- **Tier down:** Pick 3 specific changes only.
