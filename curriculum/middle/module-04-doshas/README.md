# Module 4 — Doṣas (Vāta, Pitta, Kapha)

**Age band:** Middle · **Grades:** 6–8 · **Duration:** 10 sessions × 45 min (≈ 2 weeks)

## What this module is

A self-observation framework. Indian classical medicine sorts the body's functions into three categories — *vāta* (motion), *pitta* (transformation), *kapha* (structure) — and asks students to notice how these three appear in their own daily lives. It is **not** a diagnostic system in this classroom. It is a vocabulary for noticing.

Module 3 introduced *Āyurveda* and named the three *doṣa*s briefly. This module does the deeper work: where the *doṣa*s come from in the *pañcabhūta* (Module 2), how each one looks in a person's day, how the day and the year cycle through them, and what "balance" means in practice.

## Learning objectives

By the end of this module, students will be able to:

1. **Define** *doṣa, vāta, pitta, kapha, prakṛti, vikṛti* in IAST with English meaning.
2. **Map** each *doṣa* to its two constituent *pañcabhūta* — *vāta* = *ākāśa* + *vāyu*, *pitta* = *tejas* + *ap*, *kapha* = *ap* + *pṛthvī*.
3. **Identify** three qualities of each *doṣa* (e.g. *vāta* — dry, cold, mobile) and one observable sign of each in everyday life.
4. **Distinguish** *prakṛti* (constitutional baseline) from *vikṛti* (current state), and explain why the same person can be a *vāta-pitta prakṛti* but have a *kapha vikṛti* during a cold.
5. **Apply** the framework to the daily cycle (kapha morning, pitta midday, vāta evening) and the seasonal cycle.
6. **Critique** the misuse of the framework — over-typing, self-medication, prescriptive labelling — and articulate when to consult a doctor instead.

See `_shared/glossary.md` for full entries.

## Materials checklist (whole module)

- A wall-sized three-column poster of the *tridoṣa* (Sanskrit + English + elements + qualities + signs)
- Index cards (~90) for the qualities-sort activity (Day 4)
- Self-observation worksheet (one per student, used Days 3, 5, 7)
- Six small samples for the six-tastes demo (Day 6) — jaggery (sweet), lemon (sour), salt, pepper (pungent), neem/methi (bitter), unripe banana/black tea (astringent). Check allergies.
- A clock or projector showing the dosha-clock diagram (Day 5)
- Project supplies — poster paper / cardboard / phone for video

> Each student designs a 5-day "doṣa-aware week" for themselves — sleep times, meals, and one routine activity per day — with one short paragraph explaining which *doṣa* each choice is intended to support. Presented Day 10. Full brief in `assessments/project-brief.md`.

## Safety statement (read on Day 1 and again on Day 7)

This module teaches a *framework for noticing*, not a diagnostic system. Nothing in this module should change a student's existing medical care. If a student or family has questions about an actual symptom, the answer is: see a doctor. *Doṣa* talk is wellness-language, not prescription.
