# Homework — Project Implementation — Capstone (Primary)

Light daily tasks. Each takes 5–10 minutes.

1. **Day 1 — Capstone Overview and Project Menu** · Draw about today's concept. (~5 min)
2. **Day 2 — Scope and Plan** · Draw about today's concept. (~5 min)
3. **Day 3 — Research — Day 1** · Draw about today's concept. (~5 min)
4. **Day 4 — Research — Day 2** · Draw about today's concept. (~5 min)
5. **Day 5 — Mid-Project Critique** · Draw about today's concept. (~5 min)
6. **Day 6 — Build / Make — Day 1** · Draw about today's concept. (~5 min)
7. **Day 7 — Build / Make — Day 2** · Draw about today's concept. (~5 min)
8. **Day 8 — Document** · Draw about today's concept. (~5 min)
9. **Day 9 — Rehearse Presentation** · Draw about today's concept. (~5 min)
10. **Day 10 — Project Share Day** · Draw about today's concept. (~5 min)

## Notes

- Homework is intended to consolidate what was learned in class, not introduce new content.
- If a child struggles, the parent should consult the teacher rather than push.
- Capstone project days (Module 15) may have lighter daily homework so students can focus on the project.
