# Oral Recall — Module 8 Yoga (Primary)

No paper tests. Use these oral checks.

## Day 3 — pose name recall

Class chorus:
- *"What's the mountain pose called?"* → **tāḍāsana**
- *"What's the tree pose called?"* → **vṛkṣāsana**
- *"What's the child pose called?"* → **balāsana**

## Day 5 — breath check

- *"What's the bumblebee breath called?"* → **bhrāmarī**
- *"What's the Sanskrit word for breath?"* → **prāṇa**

## Day 6 — animal walk completion

Each child completes one round = sticker.

## Day 10 — showcase = automatic sticker

Every child who teaches their pose gets a sticker.

## Optional match (Day 5 or 10)

Match the pose to the animal:

```
tāḍāsana         flamingo
vṛkṣāsana        sleeping puppy
balāsana         mountain
mārjārī          bee
bhrāmarī         cat
```

Hand back with sticker; no marks.
