# Sources — Module 2 Panchabhūta (Primary)

At this band, sources are for the teacher's awareness only — not cited in student-facing material.

## Source for the framework

- Śrīmad-Bhāgavatam, Canto 2 and Canto 3 (Sāṅkhya cosmology). Specific verses (SB 2.10.3, 3.5.36, 3.26.36, 3.26.44, 3.26.50) anchor the five-element framework.
- Available in the vidya-karana RAG cache files of the **Middle-band Module 2** (see `../../middle/module-02-panchabhuta/.rag-cache/`).

## Story sources

The "Five Friends" story used on Day 1 is original to this curriculum — it draws on the *narrative tone* of Indian children's books (Amar Chitra Katha tradition) but is not a translation of any specific tale.

## Modern science context (Day 6 sort, Day 7 walk)

Standard primary-school science vocabulary — solid, liquid, gas. No formal references needed at this band.

## For parents who ask

If a parent wants to know more about the framework, point them at the Middle-band Module 2 README. The Primary version is a deliberately simplified introduction.
