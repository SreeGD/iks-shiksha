# Oral Recall Check — Module 2 Panchabhūta (Primary)

At Primary band there are no graded paper quizzes. Use these short oral checks instead.

## Day 3 — quick check (5 min)

Whole-class chorus:

1. *"What are the five friends?"* — chorus: pṛthvī, ap, tejas, vāyu, ākāśa.
2. *"Point to something Pṛthvī in this room."* — each child points.
3. *"Point to something Ap."* — each child points.
4. *"What is Tejas?"* — answer: fire, warmth, light.
5. *"What is Vāyu?"* — answer: air, breeze.

Note any child who needs prompting — work with them in the next class's settle time.

## Day 5 — Song memory check (5 min)

After the song:

1. Cover the picture cards. *"Without looking — name all five."*
2. Ask 3 individual children, one at a time, to chant the names alone.
3. If a child can't, sing it with them, no shame.

## Day 7 — Show-of-hands self-assessment

After the walk:

1. *"Hands up if you can name all five elements without help."*
2. *"Hands up if you found an imbalance on the walk."*
3. *"Hands up if you're excited about your project."*

(No marks; just temperature check.)

## Day 10 — Sticker chart

Every child gets a sticker for the show-and-tell. The sticker is for completion, not quality.

## Optional written check (Day 5 or Day 10)

For schools that need a written record:

**Match the friend to its meaning** (draw lines):

```
pṛthvī       air
ap           space
tejas        earth
vāyu         water
ākāśa        fire / heat
```

5 marks. Don't grade — just hand back with a sticker.
