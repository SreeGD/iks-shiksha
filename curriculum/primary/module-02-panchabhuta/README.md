# Module 2 — The Five Things Everywhere (Panchabhūta)

**Age band:** Primary · **Grades:** 3–5 · **Duration:** 10 sessions × 35 min

## What children will learn

1. Name the five elements in Sanskrit and English: *pṛthvī* (earth), *ap* (water), *tejas* (fire), *vāyu* (air), *ākāśa* (space).
2. Point to one thing in their classroom or home for each element.
3. Draw a picture showing all five elements together.
4. Notice an element that is "too much" or "missing" near their home.
5. Tell a 30-second story about ONE element to a younger child.

## How this module sounds

- Short sentences. Lots of drawing.
- Sanskrit terms taught one at a time, sung as a chant.
- Story-led: each element gets a story before it gets a definition.
- No quiz scores — only stickers, oral checks, and a drawing portfolio.

## Materials checklist

- One sample object per element (rock, water jar, candle, balloon, empty box)
- Drawing paper + crayons
- 5 sorting baskets labelled with the element name + picture
- Story-time mat or carpet space

## Day-by-day overview

| Day | Title | What we do |
|-----|-------|------------|
| 1 | Five Friends Story | Story + meet the five objects |
| 2 | Earth and Water | Touch, sort, draw |
| 3 | Fire and Air | Feel warmth, blow on hand |
| 4 | Space — the Quiet Friend | Notice empty space |
| 5 | Five-Element Song | Memorise + sing the names |
| 6 | Sort It (Activity Day) | Big sorting activity with toys + objects |
| 7 | Find an Imbalance | Walk + draw a place that needs help |
| 8 | Project Day 1 | Choose one element, start drawing/making |
| 9 | Project Day 2 | Finish your project |
| 10 | Show + Tell | Each child shows their project (30 sec) |

## Project (Days 8–10)

Each child picks ONE element and makes:
- A drawing OR a 3D model (clay, recycled materials) OR a 30-second story they tell aloud.
- Required: shows the element + ONE place at home where the child notices it.

## Source

The Sanskrit names and the "five qualities" idea come from old Indian books (specifically the Bhāgavatam, written ~1500–2000 years ago, and the Sāṅkhya philosophy tradition).
