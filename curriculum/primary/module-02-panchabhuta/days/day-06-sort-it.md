# Day 6 — Sort It (Big Activity Day)

**Module:** Panchabhūta (Primary) · **Time:** 35 min

## Goal

Children sort 15+ everyday objects into the five-element trays in pairs.

## Materials

- 5 large trays (or baskets) labelled with element name + picture
- Children's bring-from-home objects (2 each = ~30+)
- 3 teacher-supplied mystery objects per pair (e.g. crayon, spoon, paper boat)
- Sticker reward chart (optional)

## The flow

### Settle + chant + song (5 min)

Daily chant + the song from Day 5.

### Sort game (20 min)

- Children work in pairs. Each pair has ~5 objects.
- They place each object on the right tray.
- For each object they say aloud (in their own words): *"This is mostly Pṛthvī because it's solid."*

After 10 min of sorting, do a class **gallery walk**: pairs walk around to see how other pairs sorted. If they disagree, they can move the object — but the original pair gets to defend it.

### Group portrait (8 min)

- The class gathers around the trays.
- Children draw a "group portrait" — a picture of all 5 trays with the objects on them.

### Show + chant (2 min)

Close with the song.

## Homework

- Tomorrow we go for a walk! Wear shoes you can walk in.

## Differentiation

- **Tier up:** Pick the one object that fits THREE elements at once. Explain to a friend.
- **Tier down:** Sort just 3 objects today. The other children can help.

## Teacher's quick notes

- Disagreements during the gallery walk are GOOD. Don't resolve them — let children argue. If they ask the teacher to decide, redirect: *"Tell me both sides. I think you're both right in different ways."*
- This is the most fun day of the module. Expect noise. Embrace it.
