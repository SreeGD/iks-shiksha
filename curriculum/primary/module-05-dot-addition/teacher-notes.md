# Teacher Notes — Module 5 Dot Addition (Primary)

## Posture

Make-10 fluency is the single skill that pays off in every later math module. This is what we're building. The dot method itself is a Middle-band topic; we *peek* at it on Day 6 to plant the seed.

## Common things to handle

1. **A child can't yet recall all the 10-pairs.** That's fine — the module is the practice. Don't shame; bring them into more games.
2. **A child gets bored ("I already know this!").** Give them the "scanning" challenge: find ALL 10-pairs in a list of 8 digits in 5 seconds.
3. **A child says "1+9 isn't always 10, what about negative numbers?"** Acknowledge — *"That's a big question for older grades. For now we're working with the counting numbers."*

## Safety

- No physical risk in this module.

## Differentiation

- **Tier up:** Grade 5 children may be ready for 2-digit + 1-digit mental addition using the Add-9 trick. Push to 47+9 = 56 by Day 5.
- **Tier down:** Grade 3 children still building 10-pair fluency need EXTRA game time. Skip the column-addition game (Day 4) and do another 10-pair game instead.

## Project oversight

- Days 7–10: children design a "make 10" game. Encourage variety — card games, board games, dice games, jumping games.
- Materials limit: only what's in the classroom. Don't expect families to provide game supplies.

## If you only have 6 days

- Day 1: Make Ten Game
- Day 2: Bonds of Ten chant + Find the Pair
- Day 3: Add-9 Trick
- Day 4: Meet the Dot + Game Day
- Day 5: Project Day
- Day 6: Game Showcase

Skip the dedicated column-addition game.
