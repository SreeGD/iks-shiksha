# Day 6 — Meet the Dot (Bindu Paddhati)

**Module:** Make Ten (Primary) · **Time:** 35 min

## Goal

Children see a demonstration of the dot method (*Bindu Paddhati*) — the technique they'll learn fully in Grade 6+.

## Materials

- Whiteboard, large
- A long column problem (4 four-digit numbers)

## The flow

### Settle + chant (5 min)

### Story (8 min)

- *"Today I'm going to show you a magic trick. Older kids — Grade 6, 7, 8 — learn this. You're going to SEE it today. We won't fully learn it until you're older."*
- *"It's called* Bindu Paddhati *— that's Sanskrit for 'dot method.'"*

### Live demo (12 min)

Write a 4-row column:
```
5728
3417
6892
4536+
```

Solve LIVE with the dot method, narrating:
- Units column: 8, 7, 2, 6.
  - 8+2 = 10! Put a dot above the 2.
  - Now I have 7 and 6 left. 7+6 = 13. That's a 10! Dot above the 6.
  - 3 is left over. Write 3 below.
  - I have 2 dots. Carry 2.
- Tens column: 2, 1, 9, 3 + the carry 2.
  - 1+9 = 10. Dot above 9.
  - 2+3+2 (carry) = 7.
  - Write 7. Carry 1.
- Hundreds: 7, 4, 8, 5 + 1.
  - Continue.
- Thousands.

Reveal: **20573**.

*"How long did that take? About a minute. Try doing this the slow way — much longer."*

### Awe + connect (8 min)

- *"You already know the trick of finding 10-pairs. That's the BIG SKILL behind this magic. When you're in Grade 6, you'll learn the dot method too. For now, keep practising those 10-pairs."*

- Class chorus of all 5 pairs.

### Show + chant (2 min)

Close.

## Homework

- Teach the chant to a younger child at home. They'll be ready when they're older.

## Differentiation

- **Tier up:** Grade 5 fast learners can try one small column problem (4 single digits) with the dot method.
- **Tier down:** Just watch the demo. Today is awe-day, not practice-day.

## Teacher's quick notes

- The "magic" is the point. Don't over-explain. Let it be mysterious.
- Don't push for understanding — just curiosity.
