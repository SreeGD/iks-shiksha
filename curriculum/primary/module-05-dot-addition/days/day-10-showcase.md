# Day 10 — Game Showcase

**Module:** Make Ten (Primary) · **Time:** 35 min

## Goal

The whole class plays each other's games in a rotation. Every child gets a sticker.

## Materials

- All children's games set up at stations
- Stickers

## The flow

### Settle + chant (3 min)

The biggest chant yet.

### Game stations (25 min)

- Each pair sets up their game at a station.
- Class rotates: 5 minutes per game station, in groups of 4.
- One member of the original pair stays at the station to teach the rules; the other plays others' games.

(Adjust based on class size — with 10 pairs and 5 minutes per station, plan 4–5 station rotations.)

### Closing ceremony (7 min)

- Whole class together.
- "Best game I played today" — quick share by 5 children.
- Stickers all round.
- Final chant — loudest yet.

### Show + chant

Done!

## Homework

- None. Module complete.

## Differentiation

- **Tier up:** A confident pair can teach two stations at once.
- **Tier down:** A shy pair can teach with just one card of rules; teacher helps.

## Teacher's quick notes

- This is celebration day. No failure. Every pair's game is a win.
- Take photos of the games and rules; send to parents.
- The skill of "rules clear enough to teach a stranger" is a real life skill — name it explicitly to the class.
