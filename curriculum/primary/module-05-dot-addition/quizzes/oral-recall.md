# Oral Recall — Module 5 (Primary)

No graded paper tests. Use these oral checks.

## Day 2 — 10-pair recall

Teacher holds up a digit. Class chorus the partner. Do 10 in 30 seconds.

8→2 / 6→4 / 3→7 / 5→5 / 9→1 / 7→3 / 4→6 / 2→8 / 1→9 / 0→10

Note: any child needing prompting works with you in next class's settle.

## Day 4 — Grouping check

Show column on board:
```
6, 3, 4, 7
```

Class shouts: "6+4=10! 3+7=10! Total = 20!"

## Day 5 — Add-9 quick fire

Teacher calls "27 plus 9!" — class shouts "36!"

10 problems in 60 seconds.

## Day 10 — Game completion sticker

Every child plays at least one game = sticker. No further assessment.

## Optional written check

Match the pair (Day 3 or 5):
```
1   6
2   5
3   4
4   3
5   2
6   1
```

Should be 1→9, 2→8 etc. for a Make-10 version. Adapt as needed.
